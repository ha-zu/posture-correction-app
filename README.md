## Getting Start

- Version
```
python -V
> Python 3.10.5
```
- Option use venv

```
python -m venv .env
```
```
source .env/bin/activate
```

- Install pkg
```
// cat requirements.txt
pip install -r requirements.txt
````

- exit venv
```
deactivate
```
